# Team Workflow

`main` is the stable demo-ready branch. `dev` is the integration branch.

1. Start from `dev` and create your feature branch.
2. Work on the feature branch and push your commits.
3. Open a PR to `dev`.
4. When `dev` is stable, open a PR from `dev` to `main`.

Rebase is optional. Normal merge or squash merge is fine.
