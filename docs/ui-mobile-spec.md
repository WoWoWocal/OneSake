# UI Mobile Spec

## Layout
- Top: Room code input and join button.
- Middle: Split panel - Left: Chat, Right: Log.
- Chat: Scrollable list of messages, input field and send button.
- Log: Scrollable list of events with sequence and type.

## Mobile Considerations
- Responsive design for small screens.
- Drawer/Sheet for log panel on mobile (future implementation).
- Touch-friendly buttons and inputs.
