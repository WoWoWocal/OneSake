# Event Log Catalog

## Log Event Types
- **SYSTEM**: General system messages (e.g., player joined).
- **DRAW**: Card draw events.
- **PLAY**: Card play actions.
- **ATTACK**: Attack declarations.
- **DEFEND**: Defense responses.
- **WIN**: Game win conditions.
- **LOSE**: Game lose conditions.

## Sequence
Each log event has a monotonic sequence number per room.
